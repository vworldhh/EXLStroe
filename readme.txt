项目介绍：这是一个专门服务于女性的电商网站，网站中分多种不同的模块，
          有衣服,化妆品,包包,美食等多种不同的模块,在网站中可以查看
女性商品的详情展示、购物车、订单下载、在线购买等模块，
网站还有后台管理系统，对商品的上架和下架,客户信息管理。
开发技术：这是一个基于传统javaweb的开发，jsp实现前端页面处理，
MVC分层开发，dbUtils工具框架的引入简化数据库语句的操作，
事务处理，ajax实现数据局部更新,mysql数据库进行数据存储
Nignx实现服务器的反向代理，Session和Cookie实现用户信息
存储以及简单的POI报表实现后台销售单和用户信息的Excel表格
打印。
